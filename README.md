# ORCA-rs

An **unofficial** Modbus frame library for the [ORCA Series](https://irisdynamics.com/products/orca-series) motor from Iris Dynamics.

## Installation

```bash
cargo add --git https://github.com/EzekielDaun/orca-rs.git
```
